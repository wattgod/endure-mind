## Practice Reminder (Day 3 Post-Module)

> Fires 3 days after each module completion.

**Subject:** Have you practiced [tool] yet?

---

It's been 3 days since you finished Module [X]. The tool was [Tool Name].

Have you used it? Not thought about it. Used it. In training. In a real session.

If yes — notice what happened. Write it down. Add it to your Evidence File.

If no — today. Not tomorrow. The gap between learning a tool and practicing it is where most athletes lose the entire investment.

## Deeper Retrieval (Day 7 Post-Module)

> Fires 7 days after module completion.

**Subject:** One week later — how did [tool] work?

---

A week ago you learned [Tool Name] in Module [X].

Two questions:
1. Have you practiced it in training? How many times?
2. What did you notice? What surprised you?

If you haven't practiced yet — that's data, not failure. The tool is still there. The lesson is still accessible. Start today.

If you have — you're building the habit. The research says it takes about 2-3 weeks of consistent practice before a mental skill starts to feel automatic. You're in the window.
