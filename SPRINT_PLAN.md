# Deliver — Build Sprint Plan

## Overview

6 sprints. Each sprint = ~1 week. Total build: ~6 weeks.
Sprints are sequenced by dependency chain — later sprints depend on earlier ones.

---

## Sprint 0: Foundation (Week 1)

**Goal:** Everything that blocks content authoring.

| Task | Deliverable | Est. Hours |
|------|-------------|------------|
| Rise 360 workspace setup | Course created, branding applied (colors, fonts, logo) | 2 |
| Design system decisions | Theme selection, accent color, heading/body fonts | 1 |
| Labeled Graphics production | Design all 13 graphics in Paper or Figma (see Appendix C) | 8-12 |
| Downloadable worksheets | Design + produce all 11 PDFs (see Appendix B) | 6-8 |
| Audio scripts | Write narration scripts for all 8 guided exercises | 4-6 |
| Email drip sequence | Write all 8 email templates (see Appendix D) | 3-4 |
| Module structure | Create 6 modules + 30 lesson shells + 6 quiz lessons in Rise | 2 |

**Sprint 0 Acceptance:**
- [ ] All 13 Labeled Graphics exported as PNG/SVG
- [ ] All 11 worksheets exported as fillable PDF
- [ ] All 8 audio scripts reviewed and approved
- [ ] Course skeleton live in Rise with correct module/lesson structure
- [ ] Branding applied and consistent

**Critical path:** Sprint 0 → Sprint 1 → Sprint 2 → Sprint 3 → Sprint 4 → Sprint 6

## Risk Register

| Risk | Likelihood | Impact | Mitigation |
|------|-----------|--------|------------|
| Content trimming takes longer than expected | High | Medium | Start with M1, establish trim template, apply to all |
| Labeled Graphics design bottleneck | Medium | High | Prioritize graphics needed for Sprint 1, batch remainder |
| Audio production delays | Medium | Medium | Course can launch without audio; add in post-launch update |
| Rise mobile rendering issues | Medium | Low | QA on mobile during Sprint 4, adjust block choices |
| Scenario blocks feel generic (stock characters) | Low | Low | Custom backgrounds compensate; focus on dialogue quality |
| Sorting block accessibility | Low | Medium | Add text alternative for all sorting activities |
| Scope creep into Storyline blocks | Low | High | Stick to native Rise blocks for v1. Storyline is a v2 enhancement |
